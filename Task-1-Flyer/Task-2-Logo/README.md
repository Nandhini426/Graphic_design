# Task 2 – Logo Design

This task involved creating a logo that represents a brand’s identity through simplicity and clarity.

## Objective
To design a logo that is minimal, scalable, and visually memorable while conveying the core values of the brand.

## Design Approach
- Followed the KISS (Keep It Simple) principle
- Used clean typography and balanced proportions
- Ensured adaptability across different platforms and sizes

## Tools Used
- Canva

## Outcome
This task strengthened my understanding of branding fundamentals and the importance of simplicity in logo design.
